Démonstration d’un composant d’onglets accessible (tablist, tab, tabpanel, aria-selected, aria-controls).

Démo

HTML

Le HTML fournit la structure sémantique du composant.

CSS

Le CSS rend visible l’onglet sélectionné et le focus clavier.

JavaScript

Le JavaScript gère la navigation clavier et l’affichage des panneaux.

Principaux éléments pour réaliser ce composant accessible

  • role="tablist"
  • role="tab"
  • role="tabpanel"
  • aria-selected
  • aria-controls
  • aria-labelledby

Spécifications d’accessibilité

Attributs ARIA

  • Le conteneur des onglets possède role="tablist".
  • Chaque onglet possède role="tab".
  • Chaque panneau possède role="tabpanel".
  • L’onglet actif possède aria-selected="true".
  • Les onglets inactifs possèdent aria-selected="false".
  • Chaque onglet pointe vers son panneau avec aria-controls.
  • Chaque panneau est relié à son onglet avec aria-labelledby.

Comportements clavier

  • Tab place le focus sur l’onglet actif.
  • ArrowRight déplace le focus vers l’onglet suivant.
  • ArrowLeft déplace le focus vers l’onglet précédent.
  • Home place le focus sur le premier onglet.
  • End place le focus sur le dernier onglet.
  • Enter ou Space active l’onglet qui a le focus.

<div class="tabs">
  <div class="tabs__list" role="tablist" aria-label="Exemples de contenus">
    <button
      type="button"
      class="tabs__tab"
      role="tab"
      id="tab-html"
      aria-selected="true"
      aria-controls="panel-html">
      HTML
    </button>

    <button
      type="button"
      class="tabs__tab"
      role="tab"
      id="tab-css"
      aria-selected="false"
      aria-controls="panel-css"
      tabindex="-1">
      CSS
    </button>

    <button
      type="button"
      class="tabs__tab"
      role="tab"
      id="tab-js"
      aria-selected="false"
      aria-controls="panel-js"
      tabindex="-1">
      JavaScript
    </button>
  </div>

  <div
    class="tabs__panel"
    role="tabpanel"
    id="panel-html"

  </div>

  <div
    class="tabs__panel"
    role="tabpanel"
    id="panel-css"

    <h3>CSS</h3>

  </div>

    <p>Le JavaScript gère la navigation clavier et l’affichage des panneaux.</p>
  </div>
</div>

JavaScript


const tabs = Array.from(document.querySelectorAll('[role="tab"]'));
const panels = Array.from(document.querySelectorAll('[role="tabpanel"]'));

function activateTab(tab) {
  tabs.forEach(currentTab => {
    const isSelected = currentTab === tab;
    const panelId = currentTab.getAttribute('aria-controls');
    const panel = document.getElementById(panelId);

    currentTab.setAttribute('aria-selected', String(isSelected));
    currentTab.tabIndex = isSelected ? 0 : -1;

    if (panel) {
      panel.hidden = !isSelected;
    }
  });
}
